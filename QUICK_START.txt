# 🚀 Quick Start Guide

Your professional resume website is ready! Follow these steps:

## Step 1: Add Your Photo ⚡
1. Find a professional headshot photo (JPG or PNG)
2. Rename it to `profile.jpg`
3. Move it to this folder (same folder as index.html)
4. Size: Roughly 200x200px works best, but any size will auto-resize

## Step 2: Test Locally 💻
1. Open `index.html` with your web browser
2. Check that:
   - All sections display correctly
   - Navigation links work
   - Photo appears
   - Mobile view looks good (resize browser)

## 🆘 Need Help?

1. **Photo not showing?**
   - Ensure file is named exactly `profile.jpg`
   - Make sure it's in the same folder as index.html

2. **Want to change colors?**
   - Open `style.css`
   - Find `:root` section at top
   - Change color codes

3. **Want to change fonts?**
   - Open `style.css`
   - Look for `font-family`

4. **GitHub deployment issues?**
   - See `GITHUB_SETUP.md`
