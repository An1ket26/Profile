# Portfolio Resume Website

A modern, responsive resume website built with HTML, CSS, and JavaScript. Hosted on GitHub Pages.

## 📋 Features

- **Modern Design**: Clean, professional interface with smooth animations
- **Fully Responsive**: Mobile-friendly design that works on all devices
- **Smooth Navigation**: Sticky navbar with smooth scrolling
- **Dark/Light Theme**: Modern gradient backgrounds
- **Optimized Performance**: Lightweight and fast-loading
- **GitHub Pages Ready**: Easy deployment to GitHub

## 📚 Sections

- **Hero Section**: Profile photo, name, contact information
- **About**: Professional summary
- **Experience**: Detailed work history with timeline
- **Projects**: Featured projects showcase
- **Skills**: Technical skills organized by category
- **Certifications**: Professional certifications and achievements
- **Education**: Educational background
- **Contact**: Contact information and social links

## 🚀 Quick Start

### 1. Add Your Photo
1. Save your profile photo as `profile.jpg` in the project folder
2. Recommended size: 200x200px
3. Supported formats: JPG, PNG, WebP

### 2. Customize Content
Edit `index.html` to customize:
- Your name and title
- Contact information
- Social links (GitHub, LinkedIn, Email, Phone)
- Experience, projects, and skills

### 3. Local Testing
Simply open `index.html` in your browser to preview locally.

## 🌐 Deploy to GitHub Pages

### Setup Instructions

1. **Create a GitHub Repository**
   - Go to [GitHub](https://github.com/new)
   - Create a new repository named `<your-username>.github.io`
   - Replace `<your-username>` with your actual GitHub username
   - Click "Create repository"

2. **Upload Files to GitHub**
   - Clone the repo: `git clone https://github.com/<your-username>/<your-username>.github.io.git`
   - Copy all portfolio files (index.html, style.css, script.js, profile.jpg) to the cloned folder
   - Open terminal in that folder and run:
     ```bash
     git add .
     git commit -m "Initial commit: Add portfolio website"
     git push origin main
     ```

3. **Enable GitHub Pages**
   - Go to your repository settings
   - Scroll to "Pages" section
   - Select `main` branch as source
   - Click "Save"

4. **Access Your Website**
   - Your site will be live at: `https://<your-username>.github.io`
   - Bookmark it and share the link!

## 🎨 Customization

### Colors
Edit the CSS variables at the top of `style.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00ccff;
    --dark-bg: #0a0e27;
    /* ... other colors ... */
}
```

### Fonts
Edit the `font-family` in `style.css`:
```css
body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}
```
